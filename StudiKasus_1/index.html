FUNDAMENTAL WEB

HTML JavaScript CSS PHP

HyperText Markup Language(HTML) adalah sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web, menampilkan Berbagi berbagi informasi di dalam sebuah penjajahan web internet dan formatinng hypertext sederahana yang ditulis dalam berkas format format ASCII agar dapat menghasilkan tampilan wujud yang terintegrasi.

CSS(Cascanding Style Sheet) adalah kependekan dari Cascanding Style Sheet. CSS merupakan kumpuulan kode-kode yang bertujuan untuk / layout halaman web supaya lebih elegan dan menarik. CSS Cascanding sy adalah sebuah teknologi internet yang direkomendasikan oleh world wide web consortium atau W3C pada tahun 1996.

>

JavaScript adalah skrip yang populer di internet dan dapat bekerja di sebagian besar penjelajah web populer seperti Internet Explorer(IE), Mozilla, Firefox, Netscape dan Opera. Kode JavaScript dapat disisipkan dalam halaman web menggunakan tag Script.

PHP adalah singkatan dari "PHP: HyperText Preprocessor", yaitu bahasa pemrograman yang digunakan secara luas untuk ppenanganan pembuatan dan pengembangan sebuah situs webdan bisa digunakan bersamaan denggan HTML. PHP diciptakan oleh Rasmus Ledoorf pertama kali tahun 1994.
